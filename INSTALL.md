## Install

Run install scripts for Ubuntu 20:
```
chmod +x setup/*.sh
./setup/prepare-install.sh
./setup/install-<target-dependency>.sh

npm install

# vue-cli bundle frontend
npm run build

# ao server at localhost:8003
npm run start
```
